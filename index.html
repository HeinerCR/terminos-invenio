Términos y Condiciones del Sistema de Reservas

Campus Heredia – Universidad Invenio. Al realizar una reserva, el usuario acepta las condiciones descritas a continuación.

Uso de Aulas

  • Destinadas exclusivamente para reposición de clases o aplicación de exámenes.
  • Solo puede reservar el profesor o profesora responsable del curso.
  • Reservas realizadas por estudiantes serán canceladas automáticamente.
  • Duración: 4 horas por bloque.
  • Para extender el tiempo deben realizarse reservas consecutivas.
Se recomienda incluir en la reserva el nombre del curso y sección.

Uso de Cubículos

  • Destinados para estudio, reuniones académicas o trabajo colaborativo.
  • Duración: 1 hora por bloque.
  • Pueden realizarse reservas consecutivas si se requiere más tiempo.
  • Debe respetarse el espacio y entregarse en las mismas condiciones.

Uso de Impresoras 3D

  • Uso exclusivo para proyectos académicos y trabajos educativos.
  • No se permite uso personal o comercial.
  • Duración: 2 horas por bloque.
  • Si requiere más tiempo deben realizarse reservas consecutivas.
  • Es responsabilidad del usuario verificar el tiempo estimado antes de reservar.

Cancelaciones y Validación

  • La administración puede cancelar reservas que no cumplan las condiciones.
  • El mal uso puede implicar suspensión temporal del sistema.
