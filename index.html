Términos y Condiciones del Sistema de Reservas

Campus Heredia – Universidad Invenio • Al realizar una reserva, el usuario acepta las condiciones descritas a continuación.

Uso de Aulas

  • Las aulas están destinadas exclusivamente para reposición de clases o aplicación de exámenes.
  • La reserva debe ser realizada únicamente por el profesor o la profesora responsable del curso.
  • Cualquier reserva realizada por estudiantes u otra persona que no sea el docente será cancelada automáticamente.
  • Duración de cada reserva: 4 horas por bloque.
  • Para extender el tiempo, deben realizarse reservas consecutivas.
Recomendación: incluir en la reserva el nombre del curso y/o sección para facilitar la validación.

Uso de Cubículos

  • Los cubículos están destinados para estudio, reuniones académicas o trabajo colaborativo.
  • Duración de cada reserva: 1 hora por bloque.
  • Puede realizar reservas consecutivas si necesita más tiempo.
  • El usuario debe respetar el espacio y dejarlo en las mismas condiciones en que fue recibido.

Uso de Impresoras 3D

  • Las impresoras 3D son de uso exclusivo para proyectos académicos, trabajos educativos y desarrollo de prototipos relacionados con cursos o actividades institucionales.
  • No se permite el uso para fines personales, comerciales o externos a la universidad.
  • Duración de cada reserva: 2 horas por bloque.
  • Si la impresión requiere más tiempo, el usuario deberá realizar 2 o 3 reservas consecutivas, según la duración estimada.
  • Es responsabilidad del usuario verificar previamente el tiempo estimado de impresión antes de confirmar la reserva.

Cancelaciones y Validación

  • La administración del campus se reserva el derecho de cancelar reservas que no cumplan con estas disposiciones.
  • El mal uso de los espacios o equipos podrá implicar la suspensión temporal del sistema de reservas para el usuario.
